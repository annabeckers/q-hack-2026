# Hero Statement

## Your developers are leaking secrets to AI — and nobody is watching.

Every day, engineers paste API keys, internal endpoints, and proprietary code into AI assistants. Cloud-based. Local. IDE-embedded. Nobody tracks what goes out, to whom, or what comes back.

**We built the intelligence layer that sees it all.**

Real-time secret detection. Malicious package scanning. Cost and compliance visibility. Deployed on a Raspberry Pi you ship to any team — no cloud required, no infrastructure changes, working in minutes.

One device. Full visibility. Zero trust required.

---

### The three things judges should remember:

1. **Secrets get caught before they leave.** Regex + entropy scanning flags API keys, credentials, and PII in every prompt — real-time, on-device.
2. **AI recommends fake packages. We catch that.** Slopsquatting detection scans AI-generated imports against known package registries.
3. **A Raspberry Pi runs the whole thing.** Ship it to a team, plug it in, get a compliance report by end of day. No vendor lock-in, no cloud dependency.
